- **BR** = Ball Roll (NOTE: Normally BS for Ball Smash, but we use BS for Back Squat, so renamed to BR)
- **C2** = Concept2 (Rower/Skierg)
- **DB** = Dumbbell
- **FR** = Foam Roller
- **GHD** = Glute-Ham Developer
- **KB** = Kettlebell
- **MB** = Medicine Ball
- **PVC** = PVC Pipe
- **RB** = Resistance Band
- **SB** = Sandbag
- **TRX** = TRX Suspension Trainer
- **WB** = Wallball

## Movement Types & Positions
- **AKBS** = American Kettlebell Swing
- **Alt.** = Alternating
- **ATG** = Ass-to-Grass
- **BSS** = Bulgarian Split Squat
- **C&P** = Clean & Press
- **C2B** = Chest to Bar
- **DL** = Deadlift
- **DUs** = Double-Unders
- **SU** = Single Unders
- **FS** = Front Squat
- **HPC** = Hang Power Clean
- **HPS** = Hang Power Snatch
- **HSPU** = Handstand Push-Up
- **HSW** = Handstand Walk
- **OH** = Overhead
- **OHP** = Overhead Press
- **OHS** = Overhead Squat
- **OTB** = Over the Box
- **PJ** = Push Jerk
- **PP** = Push Press
- **PT** = Pass Through

- **RKBS** = Russian Kettlebell Swing
- **SU** = Single-Unders (also Squat University)

## Anatomical Terms
- **Dbl** = Double
- **FHL** = Flexor Hallucis Longus
- **KoT** = Knee Over Toe
- **TFL** = Tensor Fascia Latae

## Movement Patterns & Methods
- **ATY** = (ITY) Shape of letters during movement
- **PNF** = Proprioceptive Neuromuscular Facilitation

## Programs & Resources
- **GoWOD** = Mobility app/program
- **GMB** = Gold Medal Bodies (GMB Fitness)
- **SU** = Squat University (Dr. Aaron Horschig)
